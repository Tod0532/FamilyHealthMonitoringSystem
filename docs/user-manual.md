## 1. 应用简介

### 1.1 应用概述

**家庭健康中心**是一款专为家庭设计的全方位健康管理应用，帮助您轻松管理全家人的健康数据。

### 1.2 核心功能

| 功能 | 描述 |
|------|------|
| 👨‍👩‍👧‍👦 家庭管理 | 支持添加多名家庭成员，统一管理全家健康数据 |
| 📊 数据记录 | 支持血压、血糖、心率、体温等8种健康指标记录 |
| 📈 统计分析 | 可视化图表展示健康趋势，支持多维度分析 |
| 🚨 智能预警 | 自定义预警规则，异常数据自动提醒 |
| 🩺 设备连接 | 蓝牙连接健康设备，自动同步数据 |
| 📚 健康知识 | 专业健康文章，学习养生知识 |
| 📝 健康日记 | 每日健康打卡，养成良好习惯 |
| 📤 数据导出 | 支持导出Excel/CSV，方便备份和分享 |

### 1.3 支持的健康数据类型

| 数据类型 | 单位 | 测量建议 |
|---------|------|---------|
| 血压 | mmHg | 每日早晚各测量一次 |
| 心率 | bpm/次分 | 运动后或静息状态测量 |
| 血糖 | mmol/L | 空腹或餐后2小时测量 |
| 体温 | ℃ | 发热时或日常监测 |
| 体重 | kg | 每日清晨空腹测量 |
| 身高 | cm | 定期测量记录 |
| 步数 | 步 | 每日运动量记录 |
| 睡眠 | 小时 | 每日睡眠时长记录 |

## 2. 注册与登录

### 2.1 下载安装

1. 打开手机应用商店
2. 搜索「家庭健康中心」
3. 点击「安装」按钮
4. 等待安装完成

### 2.2 新用户注册

**步骤说明**：

1. **打开应用**，点击「注册新账号」
2. **填写手机号**：输入11位手机号码
3. **获取验证码**：点击「获取验证码」按钮，输入收到的6位验证码
4. **设置密码**：
   - 密码长度6-20位
   - 必须包含字母和数字
   - 示例：`abc123456`
5. **确认密码**：再次输入相同密码
6. **设置昵称**（可选）：输入您的昵称
7. **同意协议**：勾选「我已阅读并同意用户协议和隐私政策」
8. **点击注册**：完成注册

**注意事项**：
- 手机号将作为登录账号，请填写真实可用的号码
- 密码请妥善保管，建议使用字母数字组合
- 验证码有效期为5分钟，请及时使用

### 2.3 用户登录

**步骤说明**：

1. **打开应用**
2. **输入账号**：填写注册时使用的手机号
3. **输入密码**：填写设置的密码
4. **点击登录**：完成登录进入首页

**登录示例**：
- 账号：`13800138000`
- 密码：`abc123456`

**忘记密码**？
如忘记密码，请联系客服重置或使用「忘记密码」功能找回。

### 2.4 退出登录

1. 点击底部导航栏「我的」
2. 点击右上角设置图标
3. 滑动到底部，点击「退出登录」
4. 确认退出

## 4. 家庭成员管理

### 4.1 成员页面入口

点击底部导航栏「成员」图标，进入家庭成员管理页面。

### 4.2 添加家庭成员

**步骤说明**：

1. 点击右上角 **「+」** 按钮
2. 填写成员信息：
   - **姓名**：输入成员姓名（必填）
   - **性别**：选择男/女（必填）
   - **关系**：选择与您的的关系
     - 本人
     - 配偶
     - 父亲
     - 母亲
     - 儿子
     - 女儿
     - 其他
   - **角色**：选择成员角色
     - 管理员：可管理所有成员数据
     - 普通成员：只能查看自己的数据
   - **出生日期**（可选）：选择出生日期
   - **备注**（可选）：填写备注信息
3. 点击「保存」完成添加

**注意事项**：
- 首次使用建议先添加自己作为家庭成员
- 管理员账号可以查看和管理所有成员数据
- 普通成员只能查看自己的数据

### 4.3 编辑家庭成员

**步骤说明**：

1. 在成员列表中找到要编辑的成员
2. 点击该成员卡片
3. 修改需要更新的信息
4. 点击「保存」完成编辑

### 4.4 删除家庭成员

**步骤说明**：

1. 在成员列表中找到要删除的成员
2. 向左滑动成员卡片
3. 点击「删除」按钮
4. 确认删除

**注意事项**：
- 删除成员将同时删除该成员的所有健康数据
- 此操作不可恢复，请谨慎操作
- 无法删除最后一个家庭成员

### 4.5 成员角色说明

| 角色 | 权限 |
|------|------|
| 管理员 | 查看所有成员数据、添加/编辑/删除成员、修改预警规则 |
| 普通成员 | 仅查看自己的数据、不能管理其他成员 |

### 4.6 切换当前成员

在录入健康数据时，可以通过下拉菜单选择要录入数据的成员。

## 5. 健康数据录入

### 5.1 数据录入入口

有三种方式可以进入数据录入页面：

1. **首页快捷入口**：点击首页的快捷功能按钮
2. **数据Tab页面**：点击底部「数据」Tab，点击右上角「+」按钮
3. **成员详情页**：点击成员卡片，选择「添加记录」

### 5.2 血压录入

**步骤说明**：

1. 选择数据类型：**血压**
2. 选择家庭成员
3. 输入测量数据：
   - **收缩压（高压）**：正常范围 90-140 mmHg
   - **舒张压（低压）**：正常范围 60-90 mmHg
4. 选择测量时间（默认当前时间）
5. 添加备注（可选）
6. 点击「保存」

**测量建议**：
- 每日早晚各测量一次
- 测量前静坐5分钟
- 保持坐姿，手臂与心脏同高
- 避免运动、饮食、吸烟后立即测量

**数据解读**：

| 分类 | 收缩压 | 舒张压 | 建议 |
|------|--------|--------|------|
| 低压 | < 90 | < 60 | 建议咨询医生 |
| 正常 | 90-120 | 60-80 | 继续保持 |
| 偏高 | 120-140 | 80-90 | 注意饮食，定期监测 |
| 高血压 | ≥ 140 | ≥ 90 | 建议就医检查 |

### 5.3 心率录入

**步骤说明**：

1. 选择数据类型：**心率**
2. 选择家庭成员
3. 输入心率数值（单位：次/分）
4. 选择测量时间
5. 添加备注（可选）
6. 点击「保存」

**正常范围**：
- 成人静息心率：60-100 次/分
- 运动员：可能低至40-60 次/分
- 儿童：通常比成人快

**测量时机**：
- 静息状态：早晨起床前
- 运动后：运动结束后1分钟测量
- 不适时：感到心悸、胸闷时测量

### 5.4 血糖录入

**步骤说明**：

1. 选择数据类型：**血糖**
2. 选择家庭成员
3. 输入血糖数值（单位：mmol/L）
4. 选择测量时段：
   - 空腹血糖
   - 餐后2小时
   - 随机血糖
5. 选择测量时间
6. 添加备注（可选）
7. 点击「保存」

**正常范围**（mmol/L）：

| 时段 | 正常 | 偏高 | 糖尿病 |
|------|------|------|--------|
| 空腹 | 3.9-6.1 | 6.1-7.0 | ≥ 7.0 |
| 餐后2h | < 7.8 | 7.8-11.1 | ≥ 11.1 |

**测量建议**：
- 空腹血糖：早晨未进食前测量
- 餐后血糖：进食后整2小时测量
- 定期监测，记录变化趋势

### 5.5 体温录入

**步骤说明**：

1. 选择数据类型：**体温**
2. 选择家庭成员
3. 输入体温数值（单位：℃）
4. 选择测量时间
5. 添加备注（可选）
6. 点击「保存」

**正常范围**：
- 腋下体温：36.0-37.3℃
- 口腔体温：36.3-37.2℃
- 肛门体温：36.5-37.7℃

**发热判断**：
- 低热：37.3-38.0℃
- 中热：38.1-39.0℃
- 高热：39.1-41.0℃
- 超高热：> 41.0℃

### 5.6 体重录入

**步骤说明**：

1. 选择数据类型：**体重**
2. 选择家庭成员
3. 输入体重数值（单位：kg）
4. 选择测量时间
5. 添加备注（可选）
6. 点击「保存」

**测量建议**：
- 每日固定时间测量（建议早晨起床后）
- 排空大小便后测量
- 穿着尽量轻便
- 使用同一体重秤

### 5.7 身高录入

**步骤说明**：

1. 选择数据类型：**身高**
2. 选择家庭成员
3. 输入身高数值（单位：cm）
4. 选择测量时间
5. 添加备注（可选）
6. 点击「保存」

**注意事项**：
- 儿童青少年建议每月测量一次
- 成人身高相对稳定，无需频繁测量

### 5.8 步数录入

**步骤说明**：

1. 选择数据类型：**步数**
2. 选择家庭成员
3. 输入当日步数
4. 选择日期
5. 点击「保存」

**运动建议**：
- 每日建议步数：6000-10000步
- 老年人：6000步左右即可
- 青少年：8000-10000步
- 减肥人群：10000-12000步

**数据来源**：
- 手动输入
- 手机计步器同步
- 智能手环同步

### 5.9 睡眠录入

**步骤说明**：

1. 选择数据类型：**睡眠**
2. 选择家庭成员
3. 输入睡眠时长（单位：小时）
4. 选择日期
5. 添加备注（如睡眠质量）
6. 点击「保存」

**睡眠建议**：
- 成人：7-9小时/天
- 青少年：8-10小时/天
- 儿童：9-11小时/天
- 老年人：7-8小时/天

### 5.10 批量导入数据

如需批量导入历史数据：

1. 进入数据页面
2. 点击右上角菜单
3. 选择「导入数据」
4. 选择Excel/CSV文件
5. 按照模板格式填写数据
6. 点击「导入」

## 6. 健康数据查看

### 6.1 数据页面入口

点击底部导航栏「数据」Tab，进入健康数据页面。

### 6.2 数据列表展示

数据列表默认按**时间倒序**排列，最新的数据显示在最上方。

每条数据记录显示：
- 数据类型图标和名称
- 测量数值和单位
- 测量时间
- 成员姓名
- 数据状态标识（正常/偏高/偏低/过高）

### 6.3 筛选数据

#### 6.3.1 按类型筛选

1. 点击顶部的类型筛选按钮
2. 选择要查看的数据类型
3. 列表自动刷新显示筛选结果

支持筛选的类型：
- 全部
- 血压
- 心率
- 血糖
- 体温
- 体重
- 身高
- 步数
- 睡眠

#### 6.3.2 按成员筛选

1. 点击成员筛选按钮
2. 选择要查看的成员
3. 列表自动刷新显示该成员的数据

#### 6.3.3 按日期范围筛选

1. 点击日期筛选按钮
2. 选择开始日期和结束日期
3. 列表自动刷新显示日期范围内的数据

### 6.4 查看数据详情

点击任意数据记录，可以查看详细信息：
- 完整测量数据
- 测量时间
- 所属成员
- 备注信息
- 数据状态评估

### 6.5 编辑数据

**步骤说明**：

1. 长按数据记录
2. 选择「编辑」
3. 修改需要更新的数据
4. 点击「保存」

### 6.6 删除数据

**步骤说明**：

1. 长按数据记录
2. 选择「删除」
3. 确认删除

**注意事项**：
- 删除的数据无法恢复
- 如误删，需重新录入

## 7. 健康统计分析

### 7.1 统计页面入口

1. 点击底部导航栏「数据」Tab
2. 点击右上角统计图表图标
3. 进入健康统计页面

### 7.2 统计维度

#### 7.2.1 按成员统计

选择特定家庭成员，查看该成员的健康数据统计：
- 各类数据的最新测量值
- 数据变化趋势图
- 与正常值范围对比

#### 7.2.2 按时间统计

选择时间范围，查看统计情况：
- 今日统计
- 本周统计
- 本月统计
- 自定义时间范围

#### 7.2.3 按类型统计

选择特定数据类型，查看详细统计：
- 最大值、最小值、平均值
- 数据分布情况
- 趋势变化曲线

### 7.3 图表类型

| 图表类型 | 说明 |
|---------|------|
| 折线图 | 展示数据随时间的变化趋势 |
| 柱状图 | 对比不同时间段的数据 |
| 饼图 | 展示数据分布占比 |
| 区间图 | 展示数据是否在正常范围内 |

### 7.4 血压趋势分析

血压统计页面提供：
- 收缩压趋势曲线
- 舒张压趋势曲线
- 正常范围参考线
- 异常数据标注

**查看建议**：
- 观察一周/一月趋势变化
- 注意是否有持续偏高情况
- 记录影响血压的因素（如运动、饮食、情绪）

### 7.5 血糖趋势分析

血糖统计页面提供：
- 空腹血糖趋势
- 餐后血糖趋势
- 目标范围参考线
- 血糖波动情况

**查看建议**：
- 分别查看空腹和餐后血糖
- 注意餐后血糖峰值时间
- 记录饮食与血糖关系

### 7.6 体重趋势分析

体重统计页面提供：
- 体重变化曲线
- BMI指数计算
- 理想体重范围
- 减重/增重趋势

**BMI参考**：
- 偏瘦：< 18.5
- 正常：18.5-23.9
- 超重：24-27.9
- 肥胖：≥ 28

### 7.7 导出统计报告

1. 在统计页面点击「导出」
2. 选择导出格式（PDF/图片）
3. 选择要包含的数据内容
4. 点击「生成报告」
5. 保存或分享报告

## 8. 健康预警设置

### 8.1 预警页面入口

点击底部导航栏「预警」Tab，进入健康预警页面。

### 8.2 预警类型说明

应用支持三种预警级别：

| 级别 | 图标颜色 | 说明 |
|------|---------|------|
| 信息 | 蓝色 | 提示性信息，无需特别处理 |
| 警告 | 橙色 | 数据异常，建议关注 |
| 危险 | 红色 | 严重异常，建议就医 |

### 8.3 查看预警记录

预警记录页面显示所有历史预警：
- 预警时间
- 预警类型
- 预警级别
- 触发值和阈值
- 关联成员
- 处理状态（未读/已读/已处理）

### 8.4 预警记录筛选

**标记已处理**：

**删除预警记录**：
1. 长按预警记录
2. 选择「删除」
3. 确认删除

### 8.6 预警规则设置

#### 8.6.1 进入规则设置

1. 点击预警页面右上角设置图标
2. 进入预警规则管理页面

#### 8.6.2 添加预警规则

**步骤说明**：

1. 点击「添加规则」
2. 填写规则信息：
   - **规则名称**：如「血压偏高预警」
   - **适用成员**：选择该规则应用的成员
   - **数据类型**：选择要监控的数据类型
   - **预警条件**：
     - 低于某值（下限预警）
     - 高于某值（上限预警）
     - 超出范围（区间预警）
   - **预警级别**：选择信息/警告/危险
   - **是否启用**：开启/关闭该规则
3. 点击「保存」

#### 8.6.3 预警规则示例

| 规则名称 | 数据类型 | 触发条件 | 预警级别 |
|---------|---------|---------|---------|
| 血压偏高预警 | 血压 | 收缩压≥140 或舒张压≥90 | 警告 |
| 血压危险预警 | 血压 | 收缩压≥180 或舒张压≥110 | 危险 |
| 血糖偏高预警 | 血糖 | 空腹≥7.0 或餐后2h≥11.1 | 警告 |
| 心率过快预警 | 心率 | 静息>100 次/分 | 警告 |
| 心率过缓预警 | 心率 | 静息<60 次/分 | 信息 |
| 体温发热预警 | 体温 | ≥37.3℃ | 警告 |
| 体温高热预警 | 体温 | ≥39.0℃ | 危险 |
| 体重下降预警 | 体重 | 一周下降>3kg | 信息 |

#### 8.6.4 编辑预警规则

1. 在规则列表找到要编辑的规则
2. 点击规则卡片
3. 修改规则信息
4. 点击「保存」

#### 8.6.5 删除预警规则

1. 长按规则卡片
2. 选择「删除」
3. 确认删除

#### 8.6.6 启用/禁用规则

1. 在规则列表找到目标规则
2. 点击开关按钮

## 14. 常见问题解答

### 14.1 账户相关

**Q1: 忘记密码怎么办？**

A: 可以通过以下方式找回：
1. 在登录页面点击「忘记密码」
2. 输入注册手机号
3. 获取验证码验证身份
4. 设置新密码

**Q2: 如何更换手机号？**

A:
1. 进入「我的」→「设置」→「账户安全」
2. 选择「更换手机号」
3. 验证原手机号
4. 输入并验证新手机号

**Q3: 可以同时在多台设备登录吗？**

A: 可以，同一账号可以在多台设备同时登录，但建议不超过3台。

### 14.2 数据相关

**Q4: 录入的数据可以修改吗？**

A: 可以。在数据列表中长按数据记录，选择「编辑」即可修改。

**Q5: 删除的数据能恢复吗？**

A: 删除的数据无法恢复，建议删除前确认。如需备份，请使用数据导出功能。

**Q6: 如何批量导入历史数据？**

A:
1. 进入数据页面
2. 点击「导入」按钮
3. 下载模板文件
4. 按格式填写数据
5. 上传文件完成导入

**Q7: 数据会自动备份到云端吗？**

A: 是的，数据会自动备份到云端，确保数据安全。

### 14.3 设备相关

**Q8: 为什么无法连接蓝牙设备？**

A: 请检查：
1. 手机蓝牙是否开启
2. 设备是否处于配对模式
3. 设备是否已被其他手机连接
4. 设备电量是否充足

**Q9: 支持哪些品牌的设备？**

A: 支持符合蓝牙标准的大部分健康设备，包括：
- 欧姆龙血压计
- 鱼跃血压计
- 小米手环
- 华为手环
- 云康体重秤
等支持蓝牙数据传输的设备。

**Q10: 设备数据多久同步一次？**

A:
- 自动同步：设备连接后自动同步
- 手动同步：在设备列表下拉刷新

### 14.4 预警相关

**Q11: 如何设置预警规则？**

A:
1. 进入预警页面
2. 点击右上角设置图标
3. 点击「添加规则」
4. 按提示设置规则条件

**Q12: 预警通知不及时怎么办？**

A: 请检查：
1. 手机通知权限是否开启
2. 应用是否被加入省电模式
3. 通知设置中是否开启了预警通知

### 14.5 其他问题

**Q13: 如何分享健康数据给医生？**

A:
1. 进入数据导出页面
2. 选择要导出的数据和格式
3. 导出后通过微信/邮件等方式分享

**Q14: 应用占用空间太大怎么办？**

A:
1. 进入「我的」→「设置」
2. 选择「清除缓存」
3. 清除不影响健康数据

**Q15: 如何注销账号？**

A:
1. 进入「我的」→「设置」→「账户安全」
2. 选择「注销账号」
3. 阅读注销须知
4. 确认注销

**注意**：账号注销后所有数据将被永久删除且无法恢复。
