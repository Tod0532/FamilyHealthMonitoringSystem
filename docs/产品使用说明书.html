家庭健康中心 APP
产品使用说明书

一、产品概述

1.1 产品简介

家庭健康中心是一款专为家庭设计的全方位健康管理应用，帮助您轻松管理全家人的健康数据。

1.2 核心功能

功能说明
👨‍👩‍👧‍👦 家庭管理支持添加多名家庭成员，统一管理全家健康数据
📊 数据记录支持血压、血糖、心率、体温等8种健康指标记录
📈 统计分析可视化图表展示健康趋势，支持多维度分析
🚨 智能预警自定义预警规则，异常数据自动提醒
🩺 设备连接蓝牙连接健康设备，自动同步数据
📤 数据导出支持导出Excel/CSV，方便备份和分享

1.3 支持的健康数据类型

数据类型单位测量建议
血压mmHg每日早晚各测量一次
心率bpm/次分运动后或静息状态测量
血糖mmol/L空腹或餐后2小时测量
体温℃发热时或日常监测
体重kg每日清晨空腹测量
身高cm定期测量记录
步数步每日运动量记录
睡眠小时每日睡眠时长记录

二、快速开始

2.1 下载安装

  1. 打开手机应用商店
  2. 搜索「家庭健康中心」
  3. 点击「安装」按钮
  4. 等待安装完成

2.2 注册账号

  1. 打开应用，点击「注册新账号」
  2. 填写手机号：输入11位手机号码
  3. 获取验证码：点击「获取验证码」按钮
  4. 设置密码：长度6-20位，必须包含字母和数字
  5. 设置昵称（可选）
  6. 同意用户协议和隐私政策
  7. 点击注册完成

三、功能使用指南

3.1 应用导航

应用采用底部导航栏设计，共包含5个主要Tab：

Tab图标功能
首页🏠应用主页，显示快捷入口和概览
成员👥家庭成员管理
数据📊健康数据录入和查看
预警🔔健康预警信息和规则设置
我的👤个人中心和设置

3.2 添加家庭成员

  1. 点击底部导航栏「成员」
  2. 点击右上角「+」按钮
  3. 填写作员信息：姓名、性别、关系、角色
  4. 点击「保存」完成

3.3 录入健康数据

  1. 点击首页快捷功能按钮
  2. 选择数据类型和成员
  3. 输入测量数值
  4. 选择测量时间
  5. 点击「保存」

3.4 健康统计分析

  1. 点击底部导航栏「数据」Tab
  2. 点击右上角统计图表图标
  3. 选择要查看的成员和数据类型
  4. 查看统计卡片和趋势图表

四、常见问题

问题解答
忘记密码怎么办？在登录页面点击「忘记密码」，输入手机号获取验证码后重置
删除的数据能恢复吗？删除的数据无法恢复，建议删除前使用数据导出功能备份
为什么无法连接蓝牙设备？请检查：1.手机蓝牙是否开启 2.设备是否处于配对模式 3.设备电量是否充足
如何分享健康数据给医生？进入数据导出页面，选择要导出的数据和格式，导出后通过微信/邮件分享
