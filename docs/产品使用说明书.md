## 一、产品概述

### 1.1 产品简介

**家庭健康中心**是一款专为家庭设计的全方位健康管理应用，帮助您轻松管理全家人的健康数据。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 👨‍👩‍👧‍👦 家庭管理 | 支持添加多名家庭成员，统一管理全家健康数据 |
| 📊 数据记录 | 支持血压、血糖、心率、体温等8种健康指标记录 |
| 📈 统计分析 | 可视化图表展示健康趋势，支持多维度分析 |
| 🚨 智能预警 | 自定义预警规则，异常数据自动提醒 |
| 🩺 设备连接 | 蓝牙连接健康设备，自动同步数据 |
| 📚 健康知识 | 专业健康文章，学习养生知识 |
| 📤 数据导出 | 支持导出Excel/CSV，方便备份和分享 |

### 1.3 支持的健康数据类型

| 数据类型 | 单位 | 测量建议 |
|---------|------|---------|
| 血压 | mmHg | 每日早晚各测量一次 |
| 心率 | bpm/次分 | 运动后或静息状态测量 |
| 血糖 | mmol/L | 空腹或餐后2小时测量 |
| 体温 | ℃ | 发热时或日常监测 |
| 体重 | kg | 每日清晨空腹测量 |
| 身高 | cm | 定期测量记录 |
| 步数 | 步 | 每日运动量记录 |
| 睡眠 | 小时 | 每日睡眠时长记录 |

## 二、快速开始

### 2.1 下载安装

1. 打开手机应用商店
2. 搜索「家庭健康中心」
3. 点击「安装」按钮
4. 等待安装完成

### 2.2 注册账号

**步骤说明**：

1. 打开应用，点击「注册新账号」
2. 填写手机号：输入11位手机号码
3. 获取验证码：点击「获取验证码」按钮，输入收到的6位验证码
4. 设置密码：
   - 密码长度6-20位
   - 必须包含字母和数字
   - 示例：`abc123456`
5. 设置昵称（可选）
6. 同意用户协议和隐私政策
7. 点击注册完成

### 2.3 登录应用

1. 打开应用
2. 输入注册时的手机号
3. 输入设置的密码
4. 点击登录进入首页

## 三、功能使用指南

### 3.1 应用导航结构

应用采用**底部导航栏**设计，共包含5个主要Tab：

| Tab | 图标 | 功能 |
|-----|------|------|
| 首页 | 🏠 | 应用主页，显示快捷入口和概览 |
| 成员 | 👥 | 家庭成员管理 |
| 数据 | 📊 | 健康数据录入和查看 |
| 预警 | 🔔 | 健康预警信息和规则设置 |
| 我的 | 👤 | 个人中心和设置 |

### 3.2 添加家庭成员

**步骤**：

1. 点击底部导航栏「成员」
2. 点击右上角「+」按钮
3. 填写成员信息：
   - 姓名（必填）
   - 性别（必填）
   - 与您的关系
   - 角色（管理员/普通成员）
   - 出生日期（可选）
4. 点击「保存」完成

**角色说明**：
- **管理员**：可查看和管理所有成员数据
- **普通成员**：只能查看自己的数据

### 3.3 录入健康数据

**录入方式**：

1. **首页快捷入口**：点击首页的快捷功能按钮
2. **数据Tab页面**：点击底部「数据」Tab，点击右上角「+」按钮

**血压录入示例**：

1. 选择数据类型：血压
2. 选择家庭成员
3. 输入测量数据：
   - 收缩压（高压）：正常范围 90-140 mmHg
   - 舒张压（低压）：正常范围 60-90 mmHg
4. 选择测量时间
5. 添加备注（可选）
6. 点击「保存」

**数据解读**：

| 分类 | 收缩压 | 舒张压 | 建议 |
|------|--------|--------|------|
| 低血压 | < 90 | < 60 | 建议咨询医生 |
| 正常 | 90-120 | 60-80 | 继续保持 |
| 偏高 | 120-140 | 80-90 | 注意饮食，定期监测 |
| 高血压 | ≥ 140 | ≥ 90 | 建议就医检查 |

### 3.4 查看健康数据

**数据页面功能**：

1. 点击底部导航栏「数据」Tab
2. 查看所有健康数据记录
3. 可按类型、成员、日期范围筛选

**编辑/删除数据**：

- 点击数据记录查看详情
- 选择「编辑」修改数据
- 选择「删除」删除数据（无法恢复）

### 3.5 健康统计分析

**进入统计页面**：

1. 点击底部导航栏「数据」Tab
2. 点击右上角统计图表图标
3. 进入健康统计页面

**统计功能**：

- 选择成员：查看特定成员的统计数据
- 选择数据类型：血压/血糖/心率等
- 查看统计卡片：平均值、最高值、最低值
- 查看趋势图表：近7天数据变化
- 查看数据分布：各级别数据占比
- 查看最近记录：最新5条数据记录

### 3.6 健康预警设置

**预警设置步骤**：

1. 点击底部导航栏「预警」Tab
2. 点击右上角设置图标
3. 点击「添加规则」
4. 填写规则信息：
   - 规则名称（如「血压偏高预警」）
   - 适用成员
   - 数据类型
   - 预警条件（低于/高于/超出范围）
   - 预警级别（信息/警告/危险）
5. 点击「保存」启用规则

**预警级别说明**：

| 级别 | 图标颜色 | 说明 |
|------|---------|------|
| 信息 | 蓝色 | 提示性信息，无需特别处理 |
| 警告 | 橙色 | 数据异常，建议关注 |
| 危险 | 红色 | 严重异常，建议就医 |

### 3.7 数据导出

**导出步骤**：

1. 点击底部导航栏「数据」Tab
2. 点击「数据导出」入口
3. 选择导出内容：
   - 全部/特定数据类型
   - 特定家庭成员
   - 特定时间范围
4. 选择导出格式：Excel/CSV/PDF
5. 点击「开始导出」
6. 保存或分享文件

## 四、常见问题

### Q1: 忘记密码怎么办？

**A**: 在登录页面点击「忘记密码」
1. 输入注册手机号
2. 获取验证码验证身份
3. 设置新密码

### Q2: 如何更换手机号？

**A**:
1. 进入「我的」→「设置」→「账户安全」
2. 选择「更换手机号」
3. 验证原手机号
4. 输入并验证新手机号

### Q3: 删除的数据能恢复吗？

**A**: 删除的数据无法恢复，建议删除前确认。如需备份，请使用数据导出功能。

### Q4: 为什么无法连接蓝牙设备？

**A**: 请检查：
1. 手机蓝牙是否开启
2. 设备是否处于配对模式
3. 设备是否已被其他手机连接
4. 设备电量是否充足

### Q5: 如何分享健康数据给医生？

**A**:
1. 进入数据导出页面
2. 选择要导出的数据和格式
3. 导出后通过微信/邮件等方式分享
